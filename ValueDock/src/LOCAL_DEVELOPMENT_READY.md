## üìä Environment Variables Overview

### Frontend (.env.local)

**Required:**
```env
VITE_SUPABASE_URL=https://xxx.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGc...
```

**Optional:**
```env
VITE_OPENAI_API_KEY=sk-...
VITE_GAMMA_API_KEY=...
VITE_FATHOM_PROXY_URL=https://xxx.supabase.co/functions/v1/fathom-server
```

### Backend (Supabase Secrets)

```bash
supabase secrets set OPENAI_API_KEY=sk-...
supabase secrets set FATHOM_API_KEY=...
supabase secrets set GAMMA_API_KEY=...
```

All documented in [.env.example](./.env.example)
